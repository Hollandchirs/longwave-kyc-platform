# 语言切换功能说明

## 🌐 功能概述

KYC平台现在支持**中英文双语切换**功能，用户可以在首页右上角点击语言切换按钮，在中文和英文之间无缝切换。

## ✨ 新增功能

### 1. 语言切换按钮
- **位置**：首页右上角
- **图标**：Languages 图标 + 切换文字
- **显示逻辑**：
  - 中文模式下显示 "English"
  - 英文模式下显示 "中文"

### 2. 默认语言
- **默认语言**：中文（zh）
- **可选语言**：英文（en）

### 3. 全站翻译
所有页面和内容都支持双语：

- ✅ 副标题
- ✅ 三个服务卡片（标题、描述、按钮）

#### 商业注册 (Business Incorporation)

## 🏗️ 技术实现

### 1. 语言上下文 (LanguageContext.jsx)
```javascript
- 使用 React Context API 管理全局语言状态
- 提供 language（当前语言）和 toggleLanguage（切换函数）
- 默认语言：'zh'（中文）
```

### 2. 翻译文件 (translations.js)
```javascript
- 包含所有文本的中英文版本
- 结构化组织（按页面和功能模块）
- 支持嵌套翻译对象
```

### 3. 组件集成
每个组件都使用：
```javascript
const { language } = useLanguage();
const t = translations[language].pageName;
```

## 🎯 使用说明

### 用户操作
1. 打开网站（默认显示中文）
2. 点击右上角的语言切换按钮
3. 页面立即切换为英文
4. 再次点击可切换回中文
5. 语言状态在当前会话中保持（刷新后重置为默认中文）

### 演示建议
1. **开始演示时**：使用中文，展示主要受众（国内企业出海）
2. **展示国际化能力**：中途切换到英文，展示平台的国际化特性
3. **回到中文**：结束时切换回中文，强调本土化服务
