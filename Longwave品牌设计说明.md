# Longwave 品牌设计说明

## 🎨 Logo 设计

### 设计理念

**Longwave（长波）** 的logo设计灵感来源于：
- 🌊 **波浪形态**：象征跨境、连接、流动
- 🌍 **长波传播**：寓意信息远距离传输，连接全球
- 🚀 **企业出海**：代表企业走向海外的旅程

### 视觉元素

#### 图标设计
- **圆形背景**：代表完整、全球化
- **蓝色渐变**：从浅蓝(#3B82F6)到深蓝(#1D4ED8)
  - 蓝色象征：专业、信任、稳定
  - 渐变效果：现代、科技感
- **双层波浪**：
  - 第一层波浪：实线，代表主要业务
  - 第二层波浪：半透明，代表支持服务
  - 波浪起伏：象征企业成长的起伏和持续前进

#### 文字设计
- **字体**：粗体（font-bold）
- **颜色**：蓝色渐变文字
- **副标题**：
  - 中文：企业出海服务平台
  - 英文：Corporate Services
  - 小字体，灰色，精致

## 📐 技术规格

### Logo尺寸

**首页（大尺寸）**
```
图标：48 x 48 px
公司名：24px (text-2xl)
副标题：12px (text-xs)
```

**其他页面（中尺寸）**
```
图标：40 x 40 px
公司名：20px (text-xl)
无副标题
```

### SVG代码结构

```svg
<svg width="48" height="48" viewBox="0 0 48 48">
  <!-- 渐变定义 -->
  <linearGradient id="logoGradient">
    <stop offset="0%" stopColor="#3B82F6" />
    <stop offset="100%" stopColor="#1D4ED8" />
  </linearGradient>
  
  <!-- 圆形背景 -->
  <circle cx="24" cy="24" r="24" fill="url(#logoGradient)" />
  
  <!-- 第一层波浪 -->
  <path d="..." stroke="white" opacity="1.0" />
  
  <!-- 第二层波浪 -->
  <path d="..." stroke="white" opacity="0.6" />
</svg>
```

## 💡 设计哲学

### 1. 简洁性
- 图标简单清晰，易于识别
- 不超过2种颜色（蓝色渐变 + 白色）
- 几何形状简单（圆形 + 波浪线）

### 2. 可扩展性
- SVG格式，任意缩放不失真
- 适配深色/浅色背景
- 可单独使用图标或组合使用

### 3. 记忆性
- 独特的波浪图案
- "Longwave"名称与图案呼应
- 蓝色海洋主题统一

### 4. 专业性
- 企业级视觉风格
- 现代化设计语言
- 科技感和信任感
