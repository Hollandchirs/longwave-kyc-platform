# 🚀 Longwave KYC 平台 - 快速部署指南

## 方案一：Vercel 网页部署（推荐，最简单）⭐

### 第一步：注册 Vercel 账号

1. 访问：https://vercel.com
2. 点击右上角 "Sign Up"
3. 使用 GitHub/GitLab/Bitbucket 或邮箱注册（推荐 GitHub）

### 第二步：上传项目到 GitHub

#### 2.1 在 GitHub 创建新仓库

1. 访问：https://github.com/new
2. 仓库名称：`longwave-kyc-platform`
3. 设置为 **Public**（公开）或 **Private**（私有）都可以
4. **不要**勾选 "Add a README file"
5. 点击 "Create repository"

#### 2.2 推送代码到 GitHub

在您的终端（已经在 kyc-platform 目录下）执行：

```bash
cd "/Users/hollandchirs/Desktop/KYC平台/kyc-platform"

# 关联远程仓库（替换 YOUR_USERNAME 为您的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/longwave-kyc-platform.git

# 推送代码
git branch -M main
git push -u origin main
```

> **注意**：首次推送可能需要输入 GitHub 用户名和密码（或使用 Personal Access Token）

### 第三步：在 Vercel 导入项目

1. 回到 Vercel 网站：https://vercel.com
2. 登录后，点击 "Add New..." → "Project"
3. 选择 "Import Git Repository"
4. 找到 `longwave-kyc-platform` 仓库，点击 "Import"
5. 配置项目：
   - **Framework Preset**: Vite
   - **Root Directory**: `./`
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
   - 其他保持默认
6. 点击 "Deploy"

### 第四步：等待部署完成

- 部署时间：约 1-2 分钟
- 完成后会显示：🎉 Congratulations!
- 您将获得一个网址，例如：
  - `https://longwave-kyc-platform.vercel.app`
  - `https://longwave-kyc-platform-你的用户名.vercel.app`

### 🎊 完成！

现在您可以：
- ✅ 分享链接给任何人访问
- ✅ 在任何设备上打开
- ✅ 自动获得 HTTPS 安全连接
- ✅ 免费使用，无需信用卡

## 方案二：Netlify 部署（备选方案）

### 第一步：注册 Netlify

1. 访问：https://www.netlify.com
2. 点击 "Sign Up"
3. 使用 GitHub/GitLab/Bitbucket 或邮箱注册

### 第二步：上传项目

同样需要先将代码推送到 GitHub（参考方案一的第二步）

### 第三步：在 Netlify 导入

1. 登录 Netlify
2. 点击 "Add new site" → "Import an existing project"
3. 选择 "Deploy with GitHub"
4. 授权 Netlify 访问您的 GitHub
5. 选择 `longwave-kyc-platform` 仓库
6. 配置构建设置：
   - **Build command**: `npm run build`
   - **Publish directory**: `dist`
7. 点击 "Deploy site"

### 完成！

您将获得一个网址，例如：
- `https://longwave-kyc-platform.netlify.app`

## 方案三：使用 Vercel CLI 部署（命令行方式）

### 前提条件

确保已安装 Node.js 和 npm

### 步骤

```bash
# 1. 安装 Vercel CLI（如果遇到权限问题，可以跳过此步骤）
npm install -g vercel

# 2. 进入项目目录
cd "/Users/hollandchirs/Desktop/KYC平台/kyc-platform"

# 3. 使用 npx 运行 Vercel（不需要全局安装）
npx vercel

# 4. 首次运行会要求登录：
#    - 选择登录方式（GitHub/GitLab/Bitbucket/Email）
#    - 在浏览器中完成授权

# 5. 回答配置问题：
#    ? Set up and deploy "~/Desktop/KYC平台/kyc-platform"? [Y/n] y
#    ? Which scope do you want to deploy to? [选择您的账号]
#    ? Link to existing project? [N/y] n
#    ? What's your project's name? longwave-kyc-platform
#    ? In which directory is your code located? ./
#    ? Want to modify these settings? [y/N] n

# 6. 等待部署完成，会显示网址！
```

### 生产环境部署

```bash
# 部署到生产环境
npx vercel --prod
```

## ❓ 常见问题

### Q1: 部署免费吗？

**A**: 是的！Vercel 和 Netlify 都提供免费计划，足够个人和小型项目使用。

### Q2: 部署后可以修改代码吗？

**A**: 当然可以！修改后重新推送到 GitHub，网站会自动更新。

### Q3: 如何分享给别人？

**A**: 直接把部署完成后的网址（如 `https://xxx.vercel.app`）分享给任何人，他们在浏览器中打开即可。

### Q4: 数据会丢失吗？

**A**: 目前数据存储在浏览器的 localStorage 中，每个用户的数据是独立的。如果需要共享数据，需要添加后端数据库。

### Q5: 如何删除部署？

**A**: 在 Vercel/Netlify 的控制面板中找到项目，点击 "Settings" → "Delete Project"。

### Q6: GitHub 账号是必需的吗？

**A**: 不是必需的，但强烈推荐！使用 GitHub 可以享受自动部署功能。

### Q7: 遇到部署错误怎么办？

**A**: 常见错误：
- **构建失败**：检查 `package.json` 中的依赖是否完整
- **路由 404**：确保 Vercel 的 "Output Directory" 设置为 `dist`
- **白屏**：检查浏览器控制台的错误信息
